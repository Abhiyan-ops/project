# 🚲 Hamro Ride – Online Bike Rental System
> **BCA Project | Tribhuvan University (Patan Multiple Campus)**

## 📌 Project Overview
Hamro Ride is a robust web-based management system designed to streamline the bike rental process. Developed as a core project for the Bachelor of Computer Applications (BCA), it replaces manual record-keeping with an automated, user-friendly digital platform.

### 🎯 Key Objectives
* **Accessibility:** Provide a 24/7 online booking platform.
* **Management:** Enable admins to oversee fleet and bookings in real-time.
* **Efficiency:** Reduce manual paperwork and human error.

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (Server-side logic)
* **Database:** MySQL
* **Tools:** Figma, Draw.io, XAMPP

## 👥 System Modules
* **User Module:** Registration, Login, Bike Browsing, eSewa Payment integration.
* **Admin Module:** Dashboard, Bike Inventory (CRUD), Booking Management, User Management.

## 🚀 Local Installation Guide
1. **Environment:** Install [XAMPP](https://www.apachefriends.org/).
2. **Directory:** Place the project folder inside `C:\xampp\htdocs\`.
3. **Database:** Import the provided `.sql` file into **phpMyAdmin**.
4. **Access:** Navigate to `http://localhost/your-folder-name` in your browser.
